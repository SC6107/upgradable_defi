# 前端界面设计预览

## 🎨 整体设计风格

深色主题，采用 Uniswap v2/v3 启发的现代 DeFi UI 设计

### 色彩方案
- **主色**: #FF007A (粉红色) - 用于 CTA 按钮和高亮
- **次色**: #1B1F38 (深蓝) - 背景强调
- **背景**: slate-900 (#0f172a) 到 slate-950 (#030712) 渐变
- **边框**: slate-700 (#334155)
- **文字**: 白色和 gray-400 (#9ca3af)
- **成功**: green-400 (#4ade80)
- **警告**: orange-400 (#fb923c)
- **危险**: red-400 (#f87171)

## 🔄 响应式设计

### 桌面版 (lg: 1024px+)
- 4 列网格统计卡
- 完整表格显示所有列
- 完整导航菜单

### 平板版 (md: 768px)
- 2 列网格统计卡
- 表格可能需要水平滚动
- 简化导航菜单

### 移动版 (sm: 640px)
- 1 列网格统计卡
- 表格自动水平滚动
- 标签页简化为图标

## 🎬 交互效果

### 按钮悬停

```
默认:  bg-slate-800
```

```
活跃:  bg-slate-700 text-white
```

## 🎯 关键视觉元素

### 渐变效果
- **背景**: from-slate-900 to-slate-950
- **卡片**: from-slate-700 to-slate-800
- **强调**: from-pink-500 to-purple-600

### 排版
- **标题**: bold, large (text-3xl)
- **卡片标题**: font-bold text-2xl
- **正文**: regular
- **辅文**: text-sm text-gray-400

### 间距
- **容器**: px-4 sm:px-6 lg:px-8
- **内部**: p-6
- **元素间**: gap-4 (16px)
- **部分**: mb-8 (32px)

## 🌟 高级特性

### 加载动画
```
内联旋转加载器 (spinner)
带文本: "Loading pools..."
```

### 空状态
```
带图标和消息
eg: "📭 No pools available"
```

### 可排序表头
```
点击列头排序
箭头指示方向 (↑ 升序 / ↓ 降序)
粉色高亮当前排序列
```

### 实时数据显示
- 绿色表示正向指标 (APR、供应)
- 橙色表示中性指标 (借贷利率)
- 红色表示负向指标 (利用率过高)
